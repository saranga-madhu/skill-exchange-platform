Initialize Database: Run the setup script to create the database and tables.

cd backend
node setupDatabase.js
Running the Application
1. Start Backend

	cd backend
	npm run dev
Port: 5001

2. Start Frontend

	cd skillexchange
	npm start
Port: 3000 (usually)

Verification
Automated Backend Test
We have provided a script to verify the backend API independently.

cd backend
node verify-backend.js
Manual Verification Steps
Register: Go to /register and create an account.
Login: Login with the new account. You should be redirected to Dashboard.
Dashboard: Check "My Skills" (empty initially) and "Incoming Requests" (mock data replaced with real data, so it will be empty initially).
Browse Skills: Go to Browse Skills page. You can modify filters.
Add Skill: (Note: UI for adding skill wasn't explicitly modify in this pass, but API is ready).
